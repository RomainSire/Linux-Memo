# Les Fonctions

## Déclaration d'une fonction
Première syntaxe :
```bash
maFonction ()
{
    # bloc d’instructions 
}
```
Deuxième syntaxe :
```bash
function maFonction
{
    # bloc d’instructions 
}
```

## Appel de la fonction
Il faut juste écrire son nom, comme une commande
```bash
maFonction
```
